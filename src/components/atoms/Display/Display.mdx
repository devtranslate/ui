# Display

The **Display** component is ideal for large visual highlights, such as hero sections or main calls to action. For semantic section titles and page structure, prefer the **Heading** component.

## Semantic

Currently this component offers the tags `h1`, `h2`, `h3`, `h4`, `h5`, `h6` and `span`.

> Always remember to analyze the component you intend to implement and pass the `as` prop appropriate to the context where it will be inserted. This practice is important as it helps preserve the page semantics.

<br />

## Colors

The **color** property allows you to customize the color of the component.

<Story of={DisplayStories.Colors} />

> Always use the color tokens defined in the design system, such as `gray[300]`, `blue[600]`, `red[500]`, etc. This ensures visual consistency and alignment with the brand guidelines. Avoid using arbitrary color values.
